• Coding

    Codificación

    Proceso de escribir código fuente para crear un programa de software // The process of writing source code to create a software program.

  • Programación

    Programming

    Arte y ciencia de diseñar, codificar, depurar y mantener el software. // Programming The art and science of designing, coding, debugging, and maintaining software.

  • Algoritmo

    Algorithm

    Conjunto de pasos lógicos y detallados para solucionar un problema. // A set of logical and detailed steps to solve a problem.

  • Ágil

    Agile

    Enfoque iterativo y flexible para el desarrollo de software. // An iterative and flexible approach to software development.

  • Software

    Software

    Conjunto de programas informáticos y datos asociados que realizan tareas específicas en un ordenador. // A set of computer programs and associated data that perform specific tasks on a computer.

  • Marcos

    Framework

    Conjunto de herramientas y librerías que facilitan el desarrollo de software. // A set of tools and libraries that facilitate software development.

  • Base de datos

    Database

    Sistema organizado para almacenar, acceder y gestionar información. // An organized system for storing, accessing, and managing information.

  • Desarrollador

    Developer

    Persona que escribe el código fuente para crear software. // A person who writes the source code to create software.

  • Computación en la nube

    Cloud computing

    Modelo de computación que permite acceder a recursos y aplicaciones a través de internet. // A computing model that enables access to resources and applications over the internet.

  • Integración continua

    Continuous integration

    Proceso de integrar y validar código fuente frecuentemente para detectar errores temprano. // The process of frequently integrating and validating source code to detect errors early on.

  • Despliegue

    Deployment

    Proceso de lanzar una aplicación o sistema en producción. // The process of launching an application or system into production.

  • Arquitectura de software

    Software architecture

    Diseño de un sistema de software, incluyendo componentes, interacciones y dependencias. // The design of a software system, including components, interactions, and dependencies.

  • Interfaz de usuario

    User interface

    Parte de una aplicación o sistema que permite a los usuarios interactuar con ella. // The part of an application or system that allows users to interact with it.

  • Aplicación móvil

    Mobile app

    Software diseñado para funcionar en dispositivos móviles, como smartphones y tabletas. Software designed to run on mobile devices, such as smartphones and tablets.

  • Sistema operativo

    Operating system

    Software que controla el hardware de una computadora y proporciona servicios a otros programas. // Software that controls the hardware of a computer and provides services to other programs.
